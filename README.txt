# Smart Buyer - Backend 🛒

Welcome to the Smart Buyer backend repository! This repository houses the server-side components of our innovative grocery shopping companion, Smart Buyer. Our mission is to help users make smarter shopping choices and optimize their grocery spending. 🚀

## Overview 🌟

Smart Buyer is designed to streamline the grocery shopping experience by identifying the most cost-effective stores for your specific shopping cart. With our intelligent platform, you can effortlessly create a customized shopping list, compare prices across multiple stores, and make informed decisions to save on your grocery bills.

Visit the website: [www.besmartbuyers.com](https://www.besmartbuyers.com) 🌐

## Technologies 🧰

- **Language:** Python 🐍
- **Framework:** Flask 🌐
- **Database:** PostgreSQL 🐘
- **Containerization:** Docker 🐳
- **Hosting:** Heroku (PaaS) 🚢

## Key Features 🎯

- **Custom Shopping Lists:** Create and manage shopping lists tailored to your needs. 📋
- **Price Comparison:** Compare prices from different stores for items in your cart. 💲
- **Real-time Updates:** Automated data updates every 24 hours for accurate pricing information from major supermarket chains in Israel. 🔄
- **Seamless Communication:** A custom communication protocol ensures smooth frontend-backend interaction. 📡

## Getting Started 🚀

To set up the backend locally, follow these steps:

1. Clone this repository to your local machine.
2. Install the required dependencies by running `pip install -r requirements.txt`.
3. Set up a PostgreSQL database and configure the connection in the `config.py` file.
4. Run the Flask application using `python app.py`.

## Hosting and Deployment ☁️

Backend is hosted on Heroku, a robust cloud platform. Automated data updates are scheduled to keep the information up-to-date.Use AWS Route 53 for domain management and AWS Amplify for frontend hosting and cloud services.
